kubectl create secret generic app-secret \
    --from-literal=DB_Host=mysql \
    --from-literal=DB_User=root

k create secret generic app-secret --from-file=app_secret.properties


Secrets are by default stored in base64 encoding without further encryption. For encryption, see Encrypting secret data at rest
Encoding data (when editing yaml file):
    echo -n 'mysql' | base64
Decoding:
    echo - n 'bX1rw=34' | base64 --decode

k describe secrets -> doesn't show the hash values of the secrets
k get secret app-secret -o yaml -> also shows the hash values
